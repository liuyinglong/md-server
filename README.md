## install
npm install md-http -g

## use

md-http --path="your md path" --port 80

![使用图片](https://blog.getlove.cn/img/md-http.gif)
![使用图片](https://blog.getlove.cn/img/md-http-pm2.gif)

## options

- path  
markdown文件路径 默认当前目录
- port  
端口号 默认3000
